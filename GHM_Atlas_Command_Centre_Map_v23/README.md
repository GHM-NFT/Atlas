# GHM Atlas Command Centre Map v23

v23 follows the supplied visual mock-up as the composition source of truth while retaining the deeper v20-style UX and full prototype dataset.

## Approved navigation

Top navigation:
- Overview
- Bird’s-eye
- Structured
- Block
- Waffle
- Timeline
- Register
- Dependencies
- Evidence
- Decisions

There is no separate Visual Map page. Bird’s-eye contains four internal states:
- Dependency
- Territory
- Workflow
- Risk focus

The left rail contains:
- the five core view controls
- the Territory Index
- the 11 official small gold territory shortcuts
- the five project-control shortcuts

The small gold icons are used only in the left rail.

## Overview landing page

- Atlas Command Centre introduction panel
- improved, typography-led primary actions
- compact wax-seal programme signals
- live 248-item Bird’s-eye preview
- four automatically selected live Territory Pulse cards
- condensed overview of all 11 territories
- next milestone, blocker, decision and evidence watch
- decorative Ethos footer

## Territory Pulse selection

The four cards surface distinct territories for:
- most blocked
- highest risk
- nearest milestone
- most active

Each card contains live metrics, a territory-specific waffle, the next action and drill-down controls.

## Data and drill-down

- 11 workstreams
- 248 complete prototype items
- coloured status dots
- dependency paths and risk rings
- Territory Index and detailed Territory Overview
- quick command chamber
- individual item details
- Timeline, Register, Dependencies, Evidence and Decisions
- CSV register export
