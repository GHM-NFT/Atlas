# GHM Atlas Command Centre Map v22

v22 is a controlled hybrid built around the established v20 visual and UX direction.

## Navigation
The left rail is divided into three clear groups:

1. Overview and map views
   - Overview
   - Bird's-eye
   - Structured
   - Block
   - Waffle
2. Territories
   - Opens a dedicated index page containing all 11 territories
3. Project controls
   - Timeline
   - Register
   - Dependencies
   - Evidence
   - Decisions

The 11 large workstream emblems remain as buttons in the top navigation. The small gold icon system is confined to the left navigation.

## Territory system
The Territory Index summarises:
- progress and health
- active work
- blockers and risks
- current position
- next priority
- incoming and outgoing dependencies
- review state

Each territory opens a full operational overview with:
- current position
- immediate next actions
- blockers and critical risks
- incoming and outgoing territory dependencies
- owners and workload
- evidence and decisions
- a complete item register with status filters

A translucent quick chamber remains available from the top workstream buttons and from territory labels in Bird's-eye Map.

## Bird's-eye Map
- full-stage map
- 248 coloured item dots
- dependency connections
- risk rings
- workstream labels instead of duplicate project emblems
- central Atlas Command Centre emblem
- layout and status controls
- full-screen mode

## Data
- 11 workstreams
- 248 fully populated prototype items
- complete owners, reviewers, due dates, milestones, evidence, decisions, next actions and dependencies
